“Thoughts on Progressive Enhancement”

Progressive Enhancement is accessible to web users, regardless of how old or new the browsers are. It focuses on content, rather than fancy designs and web behaviors. Not to say web designs and web behaviors are unnecessary, they are just not the main focus for browsers. Like layers, HTML, CSS, and Javascript are separated within progressive enhancement. It starts with appropriate HTML(markup language), proceeds with CSS (styling agent that brings out the feeling and design of the page), and finishes with Javascript (web behavioral). HTML, CSS, Javascript are the main components, however web users must still access the web site on mobile devices and other technology devices without Javascript.


Unlike progressive enhancement, graceful degredation can be great for web design advancement, such as fancy Javascript tools. The downside is the website quality on other technology devices. For example, my personal , Shuttured_Index Photography looks nice on desktops, but shrinks on mobile devices no matter how I fix the settings. Now, I know what the problem is and will keep that in my when I build it from scratch. To learn more about progressive enhancement and graceful degredation, you can visit WebFX or Maveneccommerce.
